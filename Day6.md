## Javascript Notes

JavaScript is a scripting or programming language that allows you to implement complex features on web pages.

It enables you to create dynamically updating content, control multimedia, animate images, and many other things.

### What Can it do?

Common Programming Features:
* Store useful values inside variables.
* Operations on pieces of text
* Running code in response to certain events occurring on a web page.

APIs can open up even more complex functionality.

There are Browser APIs and Third Party APIs.

Browser APIs enable you to input things like video and audio onto the web page.

Third Party APIs allows you to grab things like tweets from Twitter.

JavaScript is generally run in order from top to bottom.
